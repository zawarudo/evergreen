Introduction

The Filepicker component is used to select one or multiple files from the file system.

Basic multiple example

<FilePicker
  multiple
  width={250}
  marginBottom={32}
  onChange={files => console.log(files)}
  placeholder="Select the file here!"
/>

FilePicker Props

namestring
Name attribute of the input.
acceptstring | Array<string>
The accept attribute of the input.
requiredbool
When true, the file picker is required.
multiplebool
When true, accept multiple files.
disabledbool
When true, the filepicker is disabled.
capturebool
The capture attribute of the input.
heightnumber
The height of the file picker.
onChangefunc
Function called when onChange is fired
onBlurfunc
Function called when onBlur is fired
placeholderstring = 'Select a file to upload…'
Placeholder of the text input
